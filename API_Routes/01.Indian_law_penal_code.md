# ⚖️ Indian Penal & Legal API Routes Collection

---

# Basic CRUD Routes

```http
GET /api/v1/laws
→ Fetch all legal sections with pagination support

GET /api/v1/laws/:id
→ Fetch complete details of a specific legal section

POST /api/v1/laws
→ Create a new law entry in the database

PUT /api/v1/laws/:id
→ Replace entire law document with updated data

PATCH /api/v1/laws/:id
→ Partially update selected fields of a law

DELETE /api/v1/laws/:id
→ Permanently delete a law from database

GET /api/v1/laws/:id/exists
→ Check whether a specific law exists

GET /api/v1/laws/:id/summary
→ Get summarized details of a law

GET /api/v1/laws/:id/history
→ Retrieve version history of a law

PATCH /api/v1/laws/:id/archive
→ Archive a legal section

PATCH /api/v1/laws/:id/restore
→ Restore archived legal section

GET /api/v1/laws/:id/related
→ Fetch related laws and references
```
